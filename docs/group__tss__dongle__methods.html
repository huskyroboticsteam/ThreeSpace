Detailed Description

This is a collection of functions to directly communicate with a ThreeSpace Dongle.

For more information on the command functions please refer to the YOST ThreeSpace Sensors API User's Manual.

Function Documentation

◆ tss_createDongle()

TSS_EXPORT TSS_ERROR tss_createDongle ( const char *  port_name,
tss_device_id *  out_id 
)

This creates and performs initial setup on the connected 3-Space Dongle. The returned Basestation ID is used to interact with the 3-Space Dongle. When a HUB ID is created, other programs cannot use that serial port until the port is closed.

Parameters
[in]port_nameThe serial port string for the serial port to be queried.
[out]out_idA tss_device_id will be returned to represent the Dongle.
Returns
TSS_NO_ERROR(0) if successful, non zero if an error occurred.
